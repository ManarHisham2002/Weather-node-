# Weather App

A simple weather application built with Node.js, Express.js, Handlebars, Mapbox API, and WeatherAPI.

## Features

- Search weather by country or city.
- Display:
  - Location
  - Temperature
  - Weather Condition
- Error handling for invalid locations.

## Technologies Used

- Node.js
- Express.js
- Handlebars (HBS)
- Mapbox API
- WeatherAPI
- HTML
- CSS
- JavaScript

## Installation

Clone the repository:

```bash
git clone [https://github.com/YourUsername/Weather.git](https://github.com/ManarHisham2002/Weather-node-)
```

Move to the project folder:

```bash
cd Weather
```

Install dependencies:

```bash
npm install
```

Start the project:

```bash
node app.js
```

or if nodemon is installed:

```bash
nodemon app.js
```

Open your browser:

```
http://localhost:3000
```

- Mapbox Geocoding API
